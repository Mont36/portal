# GTA6Portal.ru MVP

Русскоязычный SEO-портал про GTA 6 на Next.js 15 App Router, TypeScript и Tailwind CSS.

## Возможности MVP

- Тёмная neon Vice City тема с адаптивной mobile-first версткой.
- Главная страница с hero, таймером до 19 ноября 2026, CTA, быстрыми разделами, новостями, важными фактами, RP-серверами, обсуждениями и подпиской.
- SEO-страницы: `/news`, `/gta-6-data-vyhoda`, `/gta-6-na-pk`, `/online`, `/rp`, `/servera`, `/karta`, `/personazhi`.
- Контент хранится в `lib/mock-data.ts`.
- Изображения заменены на авторские SVG-gradient placeholders из `public/placeholders`.

## Команды

```bash
npm install
npm run dev
npm run build
npm run typecheck
```
